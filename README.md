scoreloopUI_ja
author H.Ishida@MaxiGundan.
==============
2013/03/11 
Android向けオンリーの対応です。

www.scoreloop.comから提供されているSDK内の"ScoreloopUI"フォルダに追加することで、scoreloopui.jarを日本語化するものです。
対応自体は単純な"values-ja"フォルダの追加ですので、日本語端末以外は、デフォルト＝英語表示されます。
ライセンスはScoreloopUIと同様MIT Licenseとします。

・使い方
1.www.scoreloop.com にデベロッパー登録します
2.Android用最新SDK Scoreloop_core_social_android_3.1.rar をDLします
3.解凍してできるScoreloopUIフォルダ内のresフォルダに、values-ja以下を追加します
4.公式説明と同様にEclipseで、ScoreloopUIのプロジェクトをインポート、ビルドします
5.日本語対応した ScoreloopUI/bin/scoreloopui.jar が出来上がります
6.SDK同梱のデモアプリ"SLDemoUI"などで確認してみてください
